
## Development Setup

- Install [Node.js](https://nodejs.org/en/) >=v16 and [pnpm](https://pnpm.io/)
- Run `pnpm install`
- Run `pnpm dev` and visit `http://localhost:5173`

## Tech Stack
- Vue
- Vite
- VueUse
- UnoCSS
- PeerJS
- Pinia
